# CAT4D: Create Anything in 4D with Multi-View Video Diffusion Models

> **논문 정보**
> - **저자**: Rundi Wu, Ruiqi Gao, Ben Poole, Alex Trevithick, Changxi Zheng, Jonathan T. Barron, Aleksander Holynski
> - **게재**: CVPR 2025 (arXiv:2411.18613, 2024년 11월 27일 초고 공개)
> - **프로젝트 페이지**: [cat-4d.github.io](https://cat-4d.github.io/)

---

## 1. 핵심 주장 및 주요 기여 (Executive Summary)

### 1.1 핵심 주장

CAT4D는 단안(monocular) 비디오로부터 4D(동적 3D) 장면을 생성하는 방법으로, **다양한 데이터셋의 혼합으로 학습된 멀티뷰 비디오 확산 모델(multi-view video diffusion model)** 을 활용하여 임의의 카메라 포즈 및 타임스탬프에서 새로운 뷰를 합성하고, 단안 비디오를 멀티뷰 비디오로 변환함으로써 변형 가능한 3D 가우시안 표현(deformable 3D Gaussian representation)의 최적화를 통한 견고한 4D 재구성을 가능하게 합니다.

### 1.2 주요 기여 요약

| 기여 항목 | 내용 |
|---|---|
| 멀티뷰 비디오 확산 모델 | 카메라·시간 축 독립 제어 |
| 교번 샘플링 전략 | 유한 길이 제약 극복 |
| 변형 가능 3D 가우시안 재구성 | 포토메트릭 손실 기반 최적화 |
| 다양한 데이터 혼합 학습 | 정적·동적·합성 데이터 결합 |
| 외부 감독 신호 불필요 | 완전 자동화 파이프라인 |

CAT4D의 핵심은 **카메라 움직임과 장면 움직임을 분리(disentangle)** 하는 멀티뷰 비디오 확산 모델로, ① 고정 시점·시간 변화, ② 변화 시점·고정 시간, ③ 변화 시점·변화 시간의 세 가지 유형의 출력 시퀀스를 생성할 수 있습니다.

## 2. 해결하고자 하는 문제, 제안 방법, 모델 구조, 성능 및 한계

### 2.1 해결하고자 하는 문제

핵심 난제는 제한된 입력 데이터(단일 단안 비디오)로부터 시간적으로 일관되고 공간적으로 응집력 있는 뷰를 생성하는 것입니다. 멀티뷰 비디오 확산 모델 기법이 가능성을 보여주지만, **데이터 한계가 중요한 장벽**입니다. 동적 장면을 포착하는 실세계 멀티뷰 비디오 데이터는 희소하고 수집 비용이 높습니다.

단안 비디오로부터 4D 재구성을 수행하는 능력은 실제 응용에 매우 중요한데, 이는 동기화된 멀티뷰 비디오 캡처가 일반 사용자에게는 종종 비현실적이기 때문입니다.

기존 방법들의 한계:
- **Shape-of-Motion**, **MoSca** 등: 깊이 맵, 2D 트랙, 사용자 제공 전경 마스크 등 추가 감독 신호 의존
- **NeRF 기반 방법**: 동적 장면의 멀티뷰 일관성 확보 어려움
- **4D-GS**: 단독 사용 시 재구성 품질 열악

---

### 2.2 제안하는 방법 (Two-Stage Pipeline)

CAT4D는 단안 입력 비디오로부터 동적 3D 장면을 생성하는 **2단계 접근법**으로, **1단계**에서 멀티뷰 비디오 확산 모델을 이용해 단안 비디오를 멀티뷰 비디오로 변환하고, **2단계**에서 생성된 멀티뷰 비디오로부터 변형 가능한 3D 가우시안 표현을 최적화하여 동적 3D 장면을 재구성합니다.

#### 2.2.1 Stage 1: 멀티뷰 비디오 확산 모델

이 모델은 변화하는 입력 이미지, 카메라 파라미터, 타임스탬프를 조건으로 **동적 장면의 결합 분포(joint distribution)를 학습하는 확산 모델**로, 단일 단안 입력을 보다 넓은 멀티뷰 형식으로 변환하기 위한 새로운 뷰포인트와 시간 시퀀스를 생성하는 데 중추적인 역할을 합니다.

수식으로 표현하면, 모델은 주어진 입력 프레임 집합 $\mathcal{I} = \{(I_i, c_i, t_i)\}_{i=1}^{N}$ 을 조건으로, 새로운 카메라 포즈 $c^\*$ 와 타임스탬프 $t^*$ 에서의 이미지를 생성합니다:

```math
p_\theta\bigl(I^* \mid \mathcal{I},\, c^*,\, t^*\bigr)
```

여기서:
- $I^*$: 생성하려는 목표 프레임
- $c_i$: 카메라 포즈 (Plücker 좌표 등으로 인코딩)
- $t_i$: 타임스탬프
- $\theta$: 확산 모델 파라미터

확산 과정의 역방향 (denoising) 과정은 다음과 같이 정의됩니다:

$$p_\theta(\mathbf{x}_{0:T}) = p(\mathbf{x}_T)\prod_{t=1}^{T} p_\theta(\mathbf{x}_{t-1} \mid \mathbf{x}_t)$$

노이즈 예측 목표(noise prediction objective)는 다음과 같습니다:

```math
\mathcal{L}_{\text{diff}} = \mathbb{E}_{\mathbf{x}_0,\, \epsilon,\, t}\left[\left\|\epsilon - \epsilon_\theta\left(\mathbf{x}_t,\, t,\, \mathcal{I},\, c^*,\, t^*\right)\right\|^2\right]
```

#### 2.2.2 Stage 1의 데이터셋 혼합 전략 (Bespoke Dataset Mixture)

실세계 동적 장면의 멀티뷰 학습 데이터가 거의 존재하지 않기 때문에, **정적 장면의 멀티뷰 이미지**, **동적 요소를 포함한 고정 뷰포인트 비디오**, **합성 4D 데이터**, 그리고 **사전 학습된 비디오 모델과 멀티뷰 이미지 합성 모델로 증강된 추가 데이터**의 맞춤형 혼합으로 모델을 학습합니다.

| 데이터 유형 | 목적 |
|---|---|
| 정적 멀티뷰 이미지 | 멀티뷰 일관성 학습 |
| 고정 뷰포인트 동적 비디오 | 시간적 역학 학습 |
| 합성 4D 데이터 | 카메라·시간 결합 제어 학습 |
| 증강 데이터 | 데이터 희소성 보완 |

#### 2.2.3 교번 샘플링 전략 (Alternating Sampling Strategy)

모델은 자원 제약으로 인해 한 번에 **16프레임**만 생성하도록 학습되었습니다. 고품질 4D 재구성은 다수의 뷰포인트에서 많은 프레임을 필요로 하기 때문에, 확산 모델의 기본 출력 길이를 넘어 제한 없는 일관된 멀티뷰 비디오 컬렉션을 생성하는 **추가 샘플링 전략**을 제안합니다.

교번 샘플링은 **멀티뷰 샘플링과 시간적 샘플링 사이를 교번**하며 SDEdit을 사용해 이미지를 생성합니다. 멀티뷰 샘플링은 슬라이딩 윈도우 크기 3의 미디안(median)을 취하여 이미지 그리드의 각 열을 생성하고, 시간적 샘플링은 유사하게 각 행을 생성합니다. 이 병렬 처리는 생성 속도를 크게 높입니다.

이를 수식으로 표현하면, $K$ 카메라 × $L$ 타임스탬프의 이미지 그리드 $\mathbf{G} \in \mathbb{R}^{K \times L \times H \times W \times 3}$ 생성:

**멀티뷰 샘플링** (각 열 $l$ 고정, 카메라 슬라이딩 윈도우):

```math
\hat{I}_{k,l} = \text{Median}\bigl\{\epsilon_\theta\bigl(\mathbf{x}_t;\, \{c_{k'}\}_{k' \in \mathcal{W}_k},\, t_l\bigr)\bigr\}
```

**시간적 샘플링** (각 카메라 $k$ 고정, 시간 슬라이딩 윈도우):

```math
\hat{I}_{k,l} = \text{Median}\bigl\{\epsilon_\theta\bigl(\mathbf{x}_t;\, c_k,\, \{t_{l'}\}_{l' \in \mathcal{W}_l}\bigr)\bigr\}
```

#### 2.2.4 Stage 2: 변형 가능한 3D 가우시안 최적화

CAT4D는 생성된 멀티뷰 비디오와 **포토메트릭 재구성 손실(photometric reconstruction loss)** 을 사용하여 변형 가능한 3D 가우시안 표현을 최적화함으로써 동적 3D 장면을 재구성합니다.

이 모델은 포토메트릭 재구성 손실로 최적화된 **변형 가능한 3D 가우시안 표현**을 활용하며, 이를 통해 다중 뷰에 걸쳐 일관된 시각적 표현을 유지하면서 장면의 동적 측면에 유연하게 적응합니다. 최적화 전략은 정준 공간(canonical space)에서 가우시안 형태를 반복적으로 정제합니다.

포토메트릭 재구성 손실:

$$\mathcal{L}_{\text{photo}} = \sum_{k,l} \left\| \mathcal{R}\bigl(\mathcal{G}(t_l),\, c_k\bigr) - \hat{I}_{k,l} \right\|^2$$

여기서:
- $\mathcal{G}(t_l)$: 타임스탬프 $t_l$에서의 변형된 3D 가우시안 집합
- $\mathcal{R}(\cdot, c_k)$: 카메라 $c_k$에서의 렌더링 함수
- $\hat{I}_{k,l}$: 확산 모델이 생성한 멀티뷰 비디오 프레임

### 2.3 모델 구조

```
[단안 입력 비디오]
        │
        ▼
┌──────────────────────────────────┐
│  Stage 1: 멀티뷰 비디오 확산 모델   │
│  - 베이스: 텍스트→이미지 LDM        │
│  - 카메라 포즈 인코딩 (Plücker 등)  │
│  - 타임스탬프 조건부 생성           │
│  - 교번 샘플링 (K×L 그리드 확장)   │
└──────────────────────────────────┘
        │ 생성된 멀티뷰 비디오
        ▼
┌──────────────────────────────────┐
│  Stage 2: 변형 가능한 3D Gaussian  │
│  최적화                           │
│  - 정준 공간의 3D 가우시안         │
│  - 변형 필드(deformation field)   │
│  - 포토메트릭 손실로만 최적화       │
└──────────────────────────────────┘
        │
        ▼
 [4D 장면: 실시간 렌더링 가능]
```

CAT4D는 비디오 생성 모델이 아니라, **멀티뷰 비디오 확산 모델을 중심으로 구축된 동적 3D 재구성 및 생성 시스템**으로, 4D 모델 렌더링을 통한 카메라 위치·시간의 정확한 제어, 실시간 렌더링, 객체 삽입·3D 편집 등 다양한 시각 효과를 지원하는 비디오 모델이 갖지 못하는 여러 기능을 보유합니다.

### 2.4 성능 향상

DyCheck 데이터셋의 정량적 지표에서 CAT4D는 **Shape-of-Motion 및 MoSca와 대등한 경쟁력 있는 성능**을 달성하였습니다. CAT4D의 mPSNR은 22.03, mSSIM은 0.701, mLPIPS는 0.108로, Shape-of-Motion(mPSNR 22.02, mSSIM 0.699, mLPIPS 0.109) 및 MoSca(mPSNR 22.05, mSSIM 0.702, mLPIPS 0.107)와 매우 근접하거나 초과합니다.

| 방법 | mPSNR ↑ | mSSIM ↑ | mLPIPS ↓ | 추가 감독 신호 |
|---|---|---|---|---|
| **CAT4D** | **22.03** | **0.701** | **0.108** | ❌ 불필요 |
| Shape-of-Motion | 22.02 | 0.699 | 0.109 | ✅ 필요 |
| MoSca | 22.05 | 0.702 | 0.107 | ✅ 필요 |

가장 중요한 발견은 **CAT4D가 Shape-of-Motion과 MoSca에서 필수적인 깊이 맵, 2D 트랙, 사용자 제공 전경 마스크 등 추가 감독 신호에 의존하지 않고** 이에 상응하는 성능을 달성한다는 점입니다. 이는 적용 가능성과 자동화 측면에서 상당한 이점을 나타내며, CAT4D는 감독에 단순한 포토메트릭 손실만을 사용해 외부 정보 소스에 덜 의존합니다.

CAT4D는 4DiM 대비 질적·양적 지표에서 뛰어난 **장면 및 카메라의 분리 제어** 능력을 보여주고, 동적이고 비일관적인 입력 이미지 상황에서 CAT3D 대비 **희소 뷰 정적 3D 재구성에서 큰 향상**을 이뤘으며, DyCheck와 같은 어려운 데이터셋에서 최소한의 감독으로 경쟁력 있는 결과를 달성했습니다.

---

### 2.5 한계

CAT4D는 여러 한계를 가집니다: ① **확산 모델이 입력 프레임 범위를 넘는 시간적 외삽(temporal extrapolation)에 어려움**을 겪으며, 동적 객체가 가려지는 어려운 경우에 카메라 뷰포인트와 시간적 진행을 완전히 분리하지 못합니다. ② 제안된 **교번 샘플링 전략은 효과적이지만**, 이를 직접 수행하는 더 큰 규모의 멀티뷰 비디오 모델 학습이 향후 유망한 연구 방향입니다. ③ 생성된 4D 장면이 새로운 뷰포인트에서 그럴듯하게 보이지만, **복원된 3D 모션 필드는 물리적으로 정확하지 않을 수 있습니다.**

깊이나 모션 추정치 같은 감독 신호를 통합하면 결과 품질이 개선될 수 있지만, 조밀한 비디오 캡처에 대한 방법의 적용 가능성을 감소시킬 것입니다.

## 3. 모델의 일반화 성능 향상 가능성

일반화 성능은 CAT4D의 가장 핵심적인 설계 목표이자, 향후 개선 여지가 가장 큰 영역입니다.

### 3.1 다양한 데이터 혼합을 통한 일반화

멀티뷰 비디오 확산 모델 학습에 **정적 장면 멀티뷰 이미지와 동적 비디오 데이터, 합성 4D 데이터를 결합한 신중하게 선별된 혼합 데이터셋**을 사용함으로써, 카메라와 시간에 대한 분리된 제어를 가능하게 합니다.

연구자들은 합성 데이터, 데이터 증강, 또는 실제와 합성 소스의 조합에 의존하는 경우가 많습니다. **4D 장면 합성의 성공은 학습 데이터의 질과 다양성**, 그리고 카메라 포즈, 시간, 장면 역학 사이의 복잡한 관계를 효과적으로 학습하는 모델의 능력에 크게 의존합니다.

### 3.2 외부 감독 신호 없이도 가능한 일반화

시스템은 추가적인 감독 신호 없이 작동하며, **순수하게 포토메트릭 손실만으로** 최적화 과정을 안내합니다. 이는 새로운 도메인으로의 적용 가능성을 크게 높입니다.

CAT4D의 중요한 특성은 **깊이 맵, 2D 트랙, 또는 사용자 제공 전경 마스크 등 Shape-of-Motion과 MoSca에 필수적인 추가 감독 신호에 의존하지 않는다는 점**입니다. 이는 적용 가능성과 자동화 측면에서 상당한 장점을 나타내며, 외부 정보 소스에 대한 의존성을 낮춥니다.

### 3.3 일반화 성능 향상을 위한 잠재적 방향

데이터 한계가 중요한 장벽으로 남아 있습니다. **동적 장면을 포착하는 실세계 멀티뷰 비디오 데이터는 희소하고 비용이 높아** 이 부분의 확충이 일반화 성능 향상의 핵심입니다.

일반화 성능 향상을 위한 구체적 가능성:

1. **데이터 규모 확대**: 더 큰 규모의 합성+실제 혼합 데이터로 모델 학습
2. **더 긴 시퀀스 모델링**: 현재 16프레임 제한을 극복하는 **더 큰 규모의 멀티뷰 비디오 모델** 학습
3. **도메인 일반화**: 객체 중심에서 장면 수준 동적 재구성으로 확대
4. **카메라-시간 분리 개선**: 폐색(occlusion) 상황에서의 분리 능력 향상

## 4. 관련 최신 연구 비교 분석 (2020년 이후)

### 4.1 4D 생성 연구의 흐름

생성적 AI는 최근 정적 이미지·비디오 합성에서 3D 콘텐츠 생성으로 발전하여, 마침내 **시간적으로 일관된 동적 3D 자산을 합성하는 4D 생성**이 등장했습니다. 2020년 이전에는 연간 한 자리 수 논문에 불과했던 4D 생성 연구가 2024년에는 40편 이상으로 급증했습니다.

### 4.2 주요 관련 논문 비교

| 논문 | 연도 | 접근법 | CAT4D와의 차이 |
|---|---|---|---|
| **NeRF** (Mildenhall et al.) | 2020 | 정적 장면 암묵적 표현 | 동적 장면 미지원 |
| **3D Gaussian Splatting** | 2023 | 실시간 렌더링 가능 정적 표현 | 시간 축 미지원 |
| **DreamFusion** | 2023 | SDS 기반 텍스트→3D | 단일 객체·정적 |
| **CAT3D** (Gao et al.) | 2024 | 멀티뷰 확산 기반 3D 생성 | 정적 장면만 처리 |
| **SV4D** (Xie et al.) | 2024 | 멀티프레임·멀티뷰 일관성 | 객체 수준 4D 생성 |
| **Diffusion4D** | 2024 | 비디오 확산 기반 4D | 객체 중심 |
| **Shape-of-Motion** | 2024 | 2D 트랙 기반 단안 4D | 외부 감독 필요 |
| **MoSca** | 2024 | 4D 모션 스캐폴드 기반 | 외부 감독 필요 |
| **CAT4D** | 2024 | 멀티뷰 비디오 확산 + 변형 Gaussian | **장면 수준·추가 감독 불필요** |

CAT4D는 **CAT3D**(Gao et al., NeurIPS 2024)를 직접적인 선행 연구로 계승하며, 이를 정적 3D에서 동적 4D로 확장합니다.

**STAG4D**는 사전 학습된 확산 모델과 동적 3D 가우시안 스플래팅을 결합한 프레임워크로, 텍스트·이미지·비디오 등 다양한 입력에서의 4D 생성에서 새로운 최고 성능을 제시합니다.

**Diffusion4D**는 비디오 확산 모델을 통해 빠른 공간-시간적 일관성 4D 생성을 제안합니다.

## 5. 앞으로의 연구에 미치는 영향 및 연구 시 고려할 점

### 5.1 연구에 미치는 영향

#### (1) 4D 콘텐츠 생성 패러다임 전환
CAT4D는 정교한 멀티뷰 확산 전략과 혁신적인 가우시안 기반 재구성 파이프라인을 결합함으로써, **동적 환경을 캡처·시각화·상호작용하는 방식의 경계를 확장하는 다양한 응용을 위한 길을 열었습니다.**

#### (2) 하위 응용 분야 확대
이 방법은 주로 **로보틱스, 영화 제작, 비디오 게임, 증강 현실** 분야의 응용을 목표로 합니다.

#### (3) 비감독 학습 패러다임의 확산
동기화된 멀티뷰 비디오 캡처가 종종 실용적이지 않은 단안 비디오로부터 4D 재구성을 수행하는 능력은 실세계 응용에 매우 중요하며, **추가 감독 없이도 이 어려운 환경에서 뛰어난 성능을 보여주는 것은 보다 접근 가능하고 자동화된 4D 콘텐츠 생성을 향한 중요한 진보**를 의미합니다.

#### (4) 후속 연구에 대한 파급 효과

CAT4D 이후 **Diff4Splat** 등 단일 이미지에서 제어 가능하고 명시적인 4D 장면을 합성하는 피드포워드 방법이 등장하는 등, 최적화 기반 방법을 크게 능가하면서 효율성을 높이는 후속 연구들이 촉발되고 있습니다.

### 5.2 앞으로 연구 시 고려할 점

#### (1) 데이터 생태계 구축
4D 장면 합성의 성공은 **학습 데이터의 품질과 다양성**, 그리고 카메라 포즈·시간·장면 역학 사이의 복잡한 관계를 효과적으로 학습하는 모델의 능력에 크게 의존합니다. → 대규모 실세계 동적 멀티뷰 데이터셋 구축이 필수적입니다.

#### (2) 시간적 외삽 능력
확산 모델은 입력 프레임 범위를 넘는 **시간적 외삽에 어려움**을 겪으며, 동적 객체가 가려지는 경우 카메라 뷰포인트와 시간적 진행을 완전히 분리하지 못합니다. → 보다 강력한 시간적 외삽 메커니즘 연구가 필요합니다.

#### (3) 물리적 일관성
생성된 4D 장면이 새로운 뷰포인트에서 그럴듯하게 보이더라도, **복원된 3D 모션 필드는 물리적으로 정확하지 않을 수 있습니다.** → 물리 기반 제약이나 물리 시뮬레이션 통합이 중요한 연구 방향입니다.

#### (4) 대규모 모델 확장
현재 교번 샘플링 전략은 프레임 수 확장에 효과적이지만, **이를 직접 수행할 수 있는 더 큰 규모의 멀티뷰 비디오 모델 학습**이 유망한 미래 연구 영역입니다.

#### (5) 카메라 보정 문제
**카메라 보정 정보가 알 수 없거나 신뢰할 수 없는 비구조적 실세계 비디오 데이터를 처리**하기 위한 CAT4D의 적응 또는 확장이 중요한 개방 문제로 남아있습니다.

#### (6) 실시간 처리
향후 연구는 **실시간 피드백 메커니즘과 파인튜닝 방법론의 통합**을 포함할 수 있으며, 이는 다양한 시나리오에서 모델 성능을 더욱 향상시킬 것입니다.

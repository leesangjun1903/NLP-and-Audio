## 1. 핵심 주장 및 주요 기여 요약

### 🔑 핵심 주장

대규모 Text-to-Image(T2I) 확산 모델은 텍스트-이미지 생성에서 뛰어난 성능을 보이지만, 새로운 태스크를 추가하려면 모든 모달리티에 걸쳐 쌍 데이터(paired data)를 이용해 처음부터 재학습해야 하는 문제가 있다. 이 논문은 이 문제를 해결하고 **최소한의 연산으로 새로운 태스크에 모델을 확장하는 전략**을 제안한다.

실험 과정에서 확산 모델의 중간 피처 맵(intermediate feature maps)의 **분산 맵(variance maps)이 컨디셔닝의 강도를 포착**한다는 사실을 발견하였으며, 이를 사전 정보로 활용하여 새로운 모달리티 조건을 수용할 수 있도록 T2I 모델을 효율적으로 확장하는 **MaxFusion**을 제안한다.

### 📌 주요 기여 (Contributions)

| 기여 | 내용 |
|------|------|
| **Training-Free 멀티모달 생성** | 추가 학습 없이 추론 시간에만 다중 모달리티 생성 가능 |
| **분산 맵 기반 피처 융합** | 분산 맵을 기반으로 컨디셔닝 강도를 추정하는 새로운 융합 기준 제안 |
| **Plug & Play 프레임워크** | 기존 모델에 손쉽게 통합 가능 |
| **Zero-Shot 멀티모달 생성** | 단일 태스크 모델 조합만으로 제로샷 멀티모달 생성 실현 |
| **스타일 컨디셔닝 확장** | 공간적 컨디셔닝을 넘어 스타일 컨디셔닝에도 적용 가능 |

이 논문은 **학습 없는(training-free) 멀티모달 생성 솔루션**을 제안하며, 개별 단일 태스크 모델을 활용한 새로운 피처 융합 방식으로 컴포지셔널 생성을 달성한다.

## 2. 해결 문제, 제안 방법, 모델 구조, 성능 및 한계

### 2-1. 해결하고자 하는 문제 (Problem Statement)

멀티모달 생성을 위한 피처 융합 방식에서 단순히 모든 레이어의 피처를 평균화(naive averaging)하는 방법은, 피처들이 정렬(aligned)되어 있지 않을 경우 비효율적인 컨디셔닝으로 이어진다는 문제가 있다.

또한, 새로운 컨디션을 도입하기 위해 추가된 인코더의 파라미터만을 학습시켜야 하며, 기존 사전학습 가중치를 재학습하면 **파국적 망각(catastrophic forgetting)** 이 발생할 수 있다는 위험이 있다.

여러 조건으로 확장할 때 서로 다른 모달리티 간의 내재적 불균형으로 인해 각 생성 시마다 가중치 계수를 수동으로 조정해야 하는 부담이 존재한다.

---

### 2-2. 제안하는 방법 및 수식

#### 📌 핵심 관찰: 분산 맵과 컨디셔닝 강도

확산 모델에 주어진 컨디션의 표현력은 중간 레이어의 분산 맵으로 정량화할 수 있다. 서로 다른 제어 신호에서 나온 분산 맵은 특정 조건과 관련된 영역에서 더 높은 활성화를 보이며, 이는 분산 맵이 각 공간 영역에서 필요한 컨디셔닝 강도의 척도로 사용될 수 있음을 시사한다. 따라서 동일한 입력 노이즈를 서로 다른 조건 하에서 처리하는 두 모델을 비교할 때, 각 조건의 중요도는 해당 위치에서의 분산 맵 활성화 수준으로 근사할 수 있으며, 이를 통해 융합에 더 적합한 피처를 선택할 수 있다.

#### 📐 MaxFusion 수식 (공식적으로 논문에서 제시된 핵심 절차)

$j$번째 레이어, $k$번째 공간 위치에서 두 모델 $M_1$, $M_2$의 중간 피처를 각각 $f_1^{(j,k)}$, $f_2^{(j,k)}$라 하면:

**① 각 모달리티의 분산 맵 계산:**

$$\sigma^2(f_i^{(j,k)}) = \text{Var}_{c}\left(f_i^{(j,k)}\right)$$

여기서 $c$는 채널 차원이며, 각 공간 위치 $(j, k)$에서 채널 전반에 걸친 분산(variance)을 계산한다.

**② 피처 선택 기준 (MaxFusion의 핵심):**

두 모달리티 $M_1$, $M_2$에 대해 위치 $(j,k)$에서:

$$f_{\text{fused}}^{(j,k)} = \begin{cases} f_1^{(j,k)} & \text{if } \sigma^2(f_1^{(j,k)}) \geq \sigma^2(f_2^{(j,k)}) \\ f_2^{(j,k)} & \text{otherwise} \end{cases}$$

즉, 공간의 각 위치마다 **분산이 더 높은(= 컨디셔닝이 더 강한) 모델의 피처를 선택**하여 융합 피처를 구성한다.

**③ 분산 재정규화(Variance Renormalization):**

융합된 피처의 분산이 개별 피처와 일관되도록, 선택된 피처에 대해 분산 재정규화를 수행한다:

$$f_{\text{renorm}}^{(j,k)} = \frac{f_{\text{fused}}^{(j,k)} - \mu_{\text{fused}}}{\sigma_{\text{fused}}} \cdot \sigma_{\text{target}} + \mu_{\text{target}}$$

논문의 보충 자료에서는 분산 재정규화 절차 없이 MaxFusion을 구현한 변형(ablation)도 포함되어 있으며, 이를 통해 해당 절차의 효과를 검증한다.

**④ 유사 피처의 병합/비병합 (Merge-Unmerge):**

T2I 모델을 통과하는 과정에서, 중간 피처들은 각 위치에서 통합되고 결과적으로 병합된 피처가 이후 확산 모듈로 전달된다. **유사한 피처는 병합 없이 그대로 다음 레이어로 전달**되며, 유사하지 않은 중간 출력은 변경 없이 전달된다.

---

### 2-3. 모델 구조

ControlNet이나 T2I-Adapter 등 제어 확산 모델의 구조를 활용하며, 중간 레이어 출력이 T2I 모델의 특정 레이어에 추가된다. 이에 따라 **Stable Diffusion의 동일 공간 위치에 추가되는 서로 다른 모듈의 피처는 정렬(aligned)되어 있다**는 명제를 핵심 전제로 사용한다.

```
[입력: 노이즈 z_t + 텍스트 프롬프트 c_text]
         |
         ▼
┌──────────────────────────────────────────────────┐
│             Stable Diffusion (T2I Model)          │
│                                                    │
│  ControlNet(M1)    ControlNet(M2)                  │
│  (Modality 1:      (Modality 2:                    │
│   예: 깊이맵)        예: 세그멘테이션)              │
│       |                  |                         │
│       ▼                  ▼                         │
│   f1^(j,k)           f2^(j,k)                      │
│       |                  |                         │
│       └──── MaxFusion ───┘                         │
│         (분산 맵 기반 선택)                         │
│                  |                                  │
│                  ▼                                  │
│           f_fused^(j,k)                             │
│          (분산 재정규화 적용)                        │
│                  |                                  │
│                  ▼                                  │
│           다음 Diffusion 레이어                      │
└──────────────────────────────────────────────────┘
         |
         ▼
   [출력: 멀티모달 조건 기반 생성 이미지]
```

제안된 방법은 공간적 컨디셔닝을 넘어 **스타일 컨디셔닝에도 작동**하며, MaxFusion은 두 개 이상의 컨디셔닝 모달리티로도 원활하게 확장 가능하다.

---

### 2-4. 성능 향상

단일 모달리티 세그멘테이션 모델인 SPADE는 세그멘테이션 맵만으로 학습되어 엣지를 명확하게 포착하지 못하고, PITI는 복잡한 시나리오에서 이미지 품질에 어려움을 겪는다. 반면, 멀티모달 네트워크인 T2I-Adapter와 ControlNet은 대부분의 경우에서 비교적 잘 수행되지만 복잡한 시나리오에서 세부 묘사에 어려움을 겪는다.

저자들은 여러 데이터셋에 대한 실험을 통해 MaxFusion의 효과를 입증했으며, 텍스트 이외의 추가 모달리티를 통합함으로써 생성 이미지의 품질과 다양성이 크게 향상됨을 보였다. 또한 MaxFusion이 다양한 종류의 확산 모델에 적용될 수 있음을 확인하며 넓은 적용 가능성을 제시한다.

---

### 2-5. 한계

모델은 Stable Diffusion이 인간의 손과 얼굴 생성에서 가지는 한계를 그대로 물려받으며, 세맨틱 맵을 사용할 때 조건과의 불일치가 발생하는 경우도 관찰되었다. 이 한계는 ControlNet 및 T2I-Adapter의 학습 방식에서 비롯된 것이다.

또한, **조건 수가 증가할수록 컨디셔닝과 샘플링 충실도(fidelity) 사이의 트레이드오프가 발생**한다는 한계가 있다.

후속 연구(CrossControlNet)에서도 지적되었듯이, MaxFusion의 융합 전략은 디노이징 과정에서 내재하는 노이즈에 의해 쉽게 교란될 수 있으며, 공간적 조건들 간의 관계를 명시적으로 모델링하지 않아 **제어 신호 간 충돌이 있을 때 조화로운 이미지 생성에 어려움**을 겪는 경우가 있다.

## 3. 모델의 일반화 성능 향상 가능성

### 3-1. 일반화를 가능하게 하는 핵심 요소

이 논문은 확산 모델의 중간 레이어에서 분산 맵에 컨디셔닝 피처의 표현력이 내재되어 있다는 것을 발견하였으며, 이를 사전 정보로 활용해 **각각 별도로 학습된 단일 태스크 모델을 사용하여 멀티모달 생성을 위한 학습 없는(training-free) 전략**을 도입한다.

MaxFusion은 T2I 모델을 여러 태스크에 동시에 확장할 수 있는 단순하고 효율적인 피처 융합 알고리즘으로, T2I 모델에 **제로샷 멀티모달 생성 능력**을 부여한다.

### 3-2. 일반화 관점에서의 강점

MaxFusion은 텍스트-이미지 확산 모델에 멀티모달 컨디셔닝을 추가하기 위한 **플러그 앤 플레이 방식**으로, 스케치나 세그멘테이션 맵 같은 추가 입력 모달리티로도 이미지 생성이 가능하며, 기존 확산 기반 이미지 생성 모델에 쉽게 통합되도록 설계되어 있다.

플러그 앤 플레이 설계 덕분에 기존 확산 기반 이미지 생성기에 멀티모달 컨디셔닝을 쉽게 통합할 수 있으며, **전체 시스템을 재설계하지 않고도 추가 입력 모달리티 지원을 신속하게 추가**할 수 있다.

### 3-3. 제로샷 일반화

Figure 1(a)의 상단 행은 제로샷 멀티모달 생성의 경우를 보여주는데, **개별 태스크를 위해 학습된 두 모델이 오직 추론 시에만 결합**되어 멀티모달 생성을 달성한다.

MaxFusion의 융합 전략은 기존 off-the-shelf 모델에 통합되어 생성 능력을 향상시킬 수 있으며, off-the-shelf 모델을 활용한 멀티모달 생성의 효과성을 입증한다.

### 3-4. 다중 모달리티로의 확장 가능성

MaxFusion은 **두 개 이상의 컨디셔닝 모달리티로도 원활히 통합**될 수 있어, 모달리티 수에 따른 일반화 확장성이 뛰어나다.

MaxFusion과 같은 최근 연구들은 이미 공개된 강력한 사전학습 제어 모델(ControlNet 등)을 인식하고, **재학습을 피하고 대신 여러 ControlNet 브랜치를 결합하는 융합 전략**을 설계한다는 방향으로 발전하고 있다.

저자의 후속 연구인 **DiffScaler**는 단일 사전학습 확산 트랜스포머 모델을 여러 데이터셋에 걸쳐 신속하게 확장하는 데 초점을 맞추며, **최소한의 파라미터를 학습**시켜 다양한 작업에 적응하는 효율적인 스케일링 전략을 제안한다.

## 5. 향후 연구에 미치는 영향 및 고려해야 할 점

### 5-1. 앞으로의 연구에 미치는 영향

**① Training-Free 패러다임 확산**

학습 없는(training-free) 전략으로 조건부 확산 모델을 확장하는 새로운 패러다임을 제시하며, 채널 분산에 기반한 새로운 융합 패러다임을 도입하였다. 이 경량 융합 접근 방식은 기존 모델 위에 매끄럽게 통합될 수 있다.

**② 다중 모달리티 생성 연구의 기반**

여러 입력을 기반으로 이미지를 생성하는 확산 모델을 활성화함으로써, MaxFusion은 이러한 강력한 생성 AI 시스템의 창의적·실용적 응용의 새로운 가능성을 열어준다.

**③ 모델 병합 연구와의 연결**

최근 LLM 연구의 발전에서 소개된 모델 병합(model merging) 개념, 즉 동일 태스크에 대한 다중 사전학습 모델의 가중치를 결합하는 방식이 생성 모델 분야에서도 활발히 탐구되고 있으며, MaxFusion은 이러한 흐름에서 중요한 기여를 한다.

---

### 5-2. 앞으로 연구 시 고려할 점

**① 노이즈에 대한 강건성 향상**

MaxFusion의 융합 전략은 디노이징 과정의 내재적 노이즈에 취약하며, 공간 조건 간의 관계를 명시적으로 모델링하지 않아 제어 신호 충돌 시 자연스러운 이미지 생성이 어렵다는 점이 한계로 지적되고 있다. 이를 극복하는 **노이즈 강건성(noise-resilient) 융합 전략**이 필요하다.

**② 조건 간 충돌 해결**

공간적 컨디셔닝이 충돌하는 문제(conflicting spatial conditioning)를 체계적으로 해결하기 위한 연구가 필요하다.

**③ 다중 조건 시 트레이드오프 극복**

조건 수 증가에 따른 컨디셔닝과 샘플링 충실도 간의 트레이드오프를 극복하는 방법, 예컨대 **동적 가중치 조정(dynamic weighting)** 또는 **어텐션 기반 선택적 융합** 등이 연구 방향이 될 수 있다.

**④ 확산 트랜스포머(DiT)와의 통합**

최근 OpenAI의 SORA 공개와 함께 확산 트랜스포머(Diffusion Transformer)가 주목받고 있으며, 트랜스포머 기반 모델은 CNN 기반 모델에 비해 일반화 능력이 뛰어나다는 것이 밝혀졌다. MaxFusion의 분산 맵 기반 전략을 DiT 구조에 적용하는 연구가 필요하다.

**⑤ 불완전 입력에 대한 강건성**

MaxFusion이 노이즈가 많거나 불완전하거나 모호한 멀티모달 입력에 직면했을 때의 성능, 그리고 모델 복잡도나 학습 안정성 측면에서의 트레이드오프에 대한 심층 분석이 필요하다.

**⑥ 영상·3D 생성으로의 확장**

MaxFusion의 분산 맵 원리는 정적 이미지 생성에 한정되어 있으나, 비디오 및 3D 생성 모델로의 확장 가능성을 탐구할 필요가 있다.
